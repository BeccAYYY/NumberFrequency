## Getting Started

### Prerequisites

- Node.js (^20.9.2)

### Installation

```bash
# Clone the repository
git clone https://github.com/BeccAYYY/NumberFrequency.git

# Change directory
cd NumberFrequency

# Install dependencies
npm install
```

## Usage

```bash
# Run the application
npm start
```
